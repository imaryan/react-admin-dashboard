## 环境准备

克隆项目

```bash
$ git clone git@github.com:imaryan/react-admin-dashboard.git
```

安装依赖

```bash
$ npm install
```

## 启动调试

```bash
$ npm start
$ open http://localhost:8989/
```

## 构建代码

```bash
$ npm run build
```
